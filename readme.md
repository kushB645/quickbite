# 🍔 QuickBite — Full Stack Food Ordering App  

A modern **full-stack food ordering web application** built using **React.js, Express, and MongoDB**. QuickBite allows users to browse restaurants, explore menus, and manage their cart with a smooth and responsive UI.

---

## 🚀 Live Demo  
🔗 https://quickbite01.vercel.app/

---

## 📌 Features  

- 🍽️ Browse restaurants and menus  
- 🛒 Add/remove items from cart  
- ⚡ Fast and responsive UI  
- 🔄 Client-side routing using React Router  
- 🌐 REST API integration  
- 🧠 Global state management using Context API  
- 📱 Fully responsive design  
- ❌ Error handling page  

---

## 🛠️ Tech Stack  

### Frontend  
- React.js  
- JavaScript (ES6+)  
- React Router DOM  
- Context API  

### Backend  
- Node.js  
- Express.js  

### Database  
- MongoDB with Mongoose  

### Other Tools  
- Parcel (Bundler)  
- Dotenv (Environment variables)  
- CORS

## ⚙️ Installation & Setup  

### 1. Clone the repository  
git clone https://github.com/kushB645/ORDER.git  

### 2. Navigate to project folder  
cd ORDER  

### 3. Install dependencies  
npm install  

### 4. Setup environment variables  
Create a `.env` file in root:

MONGO_URI=your_mongodb_connection_string  
PORT=5000  

### 5. Run backend server  
node server/Server.js  

### 6. Start frontend  
npm start
